## Featured Projects

**[aggressor-chess-web](https://github.com/OutBlade/aggressor-chess-web)**
Chess engine written from scratch in vanilla JavaScript. Implements minimax with alpha-beta pruning, ships as a Progressive Web App with 80%+ test coverage and zero external dependencies.

**[dtZahlensysteme](https://outblade.github.io/dtZahlensysteme/)**
Number system converter (decimal, binary, octal, hexadecimal) with real-time conversion and a step-by-step learning mode. Built with React 18, TypeScript, Tailwind CSS, and Supabase. Available in German and English.

**[balatro-gba](https://github.com/GBALATRO/balatro-gba)**
Open-source Game Boy Advance game written in C. Implemented seven boss blind mechanics with round validation, post-hand resolution, and full CI/clang-format integration.

**[Sudoku Online](https://github.com/OutBlade/sudoku-online-download)**
Published Sudoku application for web and mobile.
